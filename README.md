# LogicaDeProgramacaoComVisualGPROA2026

Repositório com exercícios de lógica de programação desenvolvidos utilizando o VisualG, como parte dos estudos da disciplina PROA 2026.

Este repositório reúne algoritmos criados para praticar os conceitos fundamentais de lógica de programação, como:

Entrada e saída de dados
Estruturas condicionais (se/senão)
Estruturas de repetição (para, enquanto)
Variáveis e tipos de dados
Operadores lógicos e aritméticos

Os exercícios foram implementados utilizando o ambiente VisualG, uma ferramenta voltada para iniciantes na programação.

📁 Estrutura do repositório

Os arquivos estão organizados por exercícios, podendo conter:

Problemas resolvidos em pseudocódigo
Algoritmos com diferentes níveis de dificuldade
Práticas realizadas durante aulas ou estudos individuais

🎯 Objetivo

O objetivo deste projeto é reforçar a base em lógica de programação, essencial para o aprendizado de qualquer linguagem de programação.
